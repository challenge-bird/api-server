# 출석체크 API 서버

각 일자마다 출석체크를 기록하는 웹앱의 API 서버

## 기술 스펙
- springboot 2.6.2
- spring data jpa
- spring security
- jwt
- lombok
- mariadb

## 기능 목록
### 고객 

- [ ] 회원가입
- [ ] 회원 정보 수정
- [ ] 회원 탈퇴 
- [ ] 로그인
- [ ] 로그아웃

### 출석
- [ ] 출석 목록 조회
- [ ] 고객 출석 기록
- [ ] 고객 출석 목록 조회
